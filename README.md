# Minecraft Core

A dead simple library to store all the Minecraft utilities and data in pure Java
(intended as a core library)

## List of Bundled features

- NBT handling
- Seeds/Hashs used in Minecraft
- Jigsaw placement
- Minecraft Blocks
- Maths, Positions and Bounding Boxes
- Minecraft Versions
