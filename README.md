# MCUtils

Utilities to help when coding Minecraft Generation Librairies

## List of Bundled features

- NBT handling
- Seeds/Hashs used in Minecraft
- Jigsaw placement
- Minecraft Blocks
- Maths, Positions and Bounding Boxes
- Minecraft Versions
